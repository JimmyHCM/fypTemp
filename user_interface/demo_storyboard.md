# Demo GIF / Video Storyboard

Target length: 25–30 seconds. Capture on a mobile viewport (≤420 px wide) so touch targets read clearly.

1. **Opening frame (0–3 s)**
   - Screen: Dashboard
   - Action: Show finger cursor tapping "Gen Path" tile.
   - Callout overlay (optional): "Plan an optimized clean." (Fade out by 3 s.)

2. **Generate Path (3–10 s)**
   - Adjust overlap slider slightly to show real-time KPI updates.
   - Tap "Generate path" CTA; capture toast "Optimized path generated (94% coverage)".
   - Pause briefly on map preview emphasizing projected coverage.

3. **Mode switch to Manual (10–16 s)**
   - Use bottom nav to tap "Manual".
   - Wiggle joystick handle to show responsiveness.
   - Tap "Start recording" and show toast "Manual run recording started".

4. **Save preset (16–20 s)**
   - Tap "Stop recording" followed by "Save as preset" to trigger "Preset draft ready to name" toast.
   - Briefly pan to safety overlay to highlight warnings.

5. **End & Return (20–26 s)**
   - Tap mode tile or bottom nav to reach "End & Return".
   - Press big red "End & Return" button; capture confirmation modal.
   - Confirm and show toast "AquaSweep returning to dock".

6. **Closing frame (26–30 s)**
   - Return to Dashboard automatically (after toast) and linger on status cards updating to "Returning".

## Capture Tips
- Use screen recorder (e.g., QuickTime on macOS) with simulated touches enabled if possible.
- Trim start/end to remove setup UI.
- Export MP4 and create a short GIF variant (≤10 MB) for quick sharing.
